Evolution process

Proses evolusi bergerak dari populasi awal menuju populasi baru melalui evaluasi fitness, seleksi, crossover, dan mutasi. Setiap siklus menghasilkan generasi kandidat yang semakin mendekati konfigurasi optimal.

1. Inisialisasi populasi -> 2. Evaluasi fitness -> 3. Seleksi -> 4. Crossover -> 5. Mutasi -> 6. Generasi baru
Contoh alur satu iterasi
Populasi awal: [128, 0.15, 0.001, 16, 100] dan kandidat lain dengan variasi parameter.
Kandidat dievaluasi menggunakan validation RMSE, lalu parent terbaik dipilih untuk menghasilkan offspring baru.
Offspring kemudian dimutasi secara terbatas sebelum masuk ke generasi berikutnya.
Langkah Tahap Penjelasan
1Inisialisasi populasiMembangun kandidat solusi awal dengan kombinasi hyperparameter acak.
2Evaluasi fitnessMengukur setiap kandidat menggunakan validation RMSE.
3SeleksiMemilih individu terbaik untuk menjadi parent generasi berikutnya.
4CrossoverMenggabungkan gen parent untuk menghasilkan kombinasi baru.
5MutasiMemberi variasi kecil agar pencarian tidak terjebak pada solusi lokal.
6Generasi baruMengevaluasi hasil offspring dan mengulangi proses sampai konvergen.
